# Simple Calculator

## Description
This is a simple command-line calculator built using Python. It performs basic arithmetic operations and includes input validation, divide-by-zero handling, and a menu for repeated calculations.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Handles invalid input
- Prevents division by zero
- Menu-driven interface
- Uses functions for each operation

## Technologies Used
- Python 3

## How to Run

1. Clone the repository:
   bash
   git clone https://github.com/YourUsername/Syntexhub_SimpleCalculator.git
   

2. Open the project folder.

3. Run the program:
   bash
   python Simple Calculator.py
